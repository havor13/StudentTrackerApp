**Course:** CSE 310

## Overview
StudentTrackerApp is an Android application built with **Kotlin** and **Jetpack Compose**.  
It allows users to input student names, save them using **DataStore persistence**, and view the stored list on a second screen.  
This project demonstrates mobile app development concepts including state management, navigation, and persistent storage.

---

## Learning Goals
- Practice Kotlin programming in Android Studio.
- Learn Jetpack Compose for declarative UI.
- Implement persistent storage with DataStore.
- Manage navigation between multiple screens.
- Apply professional project structure and documentation.

---

## Features
- Input screen for adding student names.
- Save logic using DataStore.
- Navigation between screens.
- Display of stored student list with LazyColumn.
- Clean UI with padding and spacing.
